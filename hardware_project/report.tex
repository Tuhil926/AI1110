\section{Abstract}
In this project, we made a random number generator using shift registers.
\section{Components used}
\begin{enumerate}
	\item{Breadboard}
	\item{Seven segment display}
	\item{Decoder}
	\item{2 Flip Flop IC's}
	\item{X-OR GATE}
	\item{555 IC}
	\item{Resistors(1K, 1M)}
	\item{Capacitors(100nF, 10nF)}
	\item{jumper wires}
\end{enumerate}
\section{Procedure}
\begin{itemize}
	\item We generate the clock signal for the flip flops by using the 555 timer IC and making the respective connections as shown in the figure \ref{fig:timer}.
	\begin{figure}[h]
        	\centering
        	\includegraphics{images/timer.png}
        	\caption{555 Timer IC}
        	\label{fig:timer}
	\end{figure}
\item Next, we used the flip flops to make a shift register with 4 bits, $Q_0, Q_1, Q_2,Q_3$, and the inputs being $D_0, D_1, D_2, D_3$. This is shown in figure \ref{fig:block}
	\begin{figure}[h]
                \centering
                \includegraphics{images/block_dia.png}
                \caption{Shift register}
                \label{fig:block}
        \end{figure}
\item Then the XOR gate is connected such that $D_0 = Q_0 \oplus Q_3$.
\item Then, the four bits are connected to the decoder IC, which is connected to the 7-segment display.
\end{itemize}
